    # Radio Attribute

    属性 说明 类型 可选值 默认值
    value 选项的唯一值 String，Number -
    v-model 动态绑定选中的选项唯一值 String，Number -
    datas 选项数据组 Array -
    valueKey 自定义选项的值key String value
    labelKey 自定义选项的标题key String label
    disabled 是否禁用 Boolean false
